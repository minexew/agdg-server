
Note: "hash" means sha3_224 (SHA-3 224 in binary)

CLIENT -> SERVER
================

01 Client Hello

	hash token

	Reply: Server Hello

02 Enter World

	string characterName

	Reply: Load Zone

03 Zone Loaded

	Reply: Zone Intro

04 Player Movement

	vec3 pos
	vec3 dir
	vec3 velocity

30 Chat Say

	string text

SERVER -> CLIENT
================

01 Server Hello

	uint32 numCharacters

	times numCharacters:
		string characterName

02 Load Zone

	string zoneName
	hash zoneRef
	vec3 playerPos
	vec3 playerDir

03 Zone State

	uint32 numEntities

	times numEntities:
		uint32 eid
		uint32 flags
		string name
		vec3 pos
		vec3 dir

20 Entity Spawn

	uint32 eid
	uint32 flags
	string name
	vec3 pos
	vec3 dir

21 Entity Despawn

	uint32 eid

22 Entity Update

	uint32 eid
	vec3 pos
	vec3 dir
	vec3 velocity

30 Chat Say

	uint32 eid
	string text
